Flashing Instructions

Follow these steps for a clean install. Keep all files in your platform-tools folder.

Downloads

ROM Zip PixelOS_xaga.zip boot.img Kernel boot image vendor_boot.img Vendor boot image

Flash Steps

  1. Reboot to bootloader

    adb reboot bootloader

  2. Flash boot image

    fastboot flash boot boot.img

  3. Flash vendor boot image

    fastboot flash vendor_boot vendor_boot.img

  4. Boot recovery and sideload ROM

    Do not use fastboot reboot recovery on xaga. Enter recovery using the safe key combo from the bootloader menu.

    adb sideload PixelOS_xaga.zip

  5. Boot system

    adb reboot
